Numpy & pandas

Python м–ём–ҙлҠ” мҲҳм№ҳ кі„мӮ°мқ„ мң„н•ҙ л””мһҗмқёлҗҳм§Җ м•Ҡм•ҳкё° л•Ңл¬ём—җ, лҚ°мқҙн„° л¶„м„қм—җ лҢҖн•ң нҡЁмңЁм Ғмқҙкі  л№ лҘё кі„мӮ°мқҙ мҡ”кө¬лҗҳл©ҙм„ң C/C++мқҙлқјлҠ” м–ём–ҙлЎң кө¬нҳ„лҗң NumPy (Numerical Python)к°Җ нғ„мғқн•ҳмҳҖкі , Python мғқнғңкі„ м•Ҳм—җ нҶөн•©лҗҳм—ҲмқҢ. кё°ліём ҒмңјлЎң Python м–ём–ҙ м•Ҳм—җ мғҲлЎңмҡҙ м–ём–ҙлқјкі  ліј мҲҳ мһҲмқҢ. лҚ°мқҙн„° мӮ¬мқҙм–ёмҠӨм—җм„ңмқҳ лҢҖл¶Җл¶„мқҳ кі„мӮ°мқҖ NumPyмқҳ ndarray (n-dimensioal array)мҷҖ мҲҳн•ҷм Ғ operatorл“Өмқ„ нҶөн•ҙ кі„мӮ°лҗЁ.

лҚ°мқҙн„° мӮ¬мқҙм–ёмҠӨк°Җ л°ңм „н•Ём—җ л”°лқј лӢЁмқјн•ң floating-point numberл“Өмқ„ м„ұл¶„мңјлЎңн•ҳлҠ” arrayл“Өмқҳ кі„мӮ°м—җм„ң лІ—м–ҙлӮҳ м№јлҹјлі„лЎң лӢӨлҘё лҚ°мқҙн„° нғҖмһ…(string, integer, object..)мқ„ нҸ¬н•Ён•ҳлҠ” tabularнҳ•нғңмқҳ лҚ°мқҙн„°лҘј нҡЁмңЁм ҒмңјлЎң мІҳлҰ¬н•ҙм•ј н•  н•„мҡ”м„ұмқҙ лӮҳнғҖлӮ¬кі , мқҙлҘј лӢӨлЈ° мҲҳ мһҲлҠ” мғҲлЎңмҡҙ м–ём–ҙлҘј NumPy мң„м—җ к°ңл°ңн•ң кІғмқҙ pandasмһ„. мқҙлҠ” кё°ліём ҒмңјлЎң Wes Mckinneyм—җ мқҳн•ҙ лҸ…мһҗм ҒмңјлЎң к°ңл°ңмқҙ мӢңмһ‘лҗҳм—Ҳмңјл©°, л””мһҗмқём ҒмңјлЎң л¶Ҳл§ҢмЎұмҠӨлҹ¬мҡҙ м җмқҙ м§Җм Ғлҗҳкі лҠ” мһҲмңјлӮҳ лҚ°мқҙн„° мӮ¬мқҙм–ёмҠӨмқҳ кё°ліём Ғмқё м–ём–ҙк°Җ лҗҳм—ҲмқҢ.

NumPy

  • мҲҳн•ҷм Ғ symbolic м—°мӮ°м—җ лҢҖн•ң кө¬нҳ„мқҙлқјкі  ліј мҲҳ мһҲмңјл©°,

  • н–үл ¬(matrix) лҳҗлҠ” лІЎн„°(vector)лҘј ndarray (n-dimensional array)мқҙлқјлҠ” мқҙлҰ„мңјлЎң кө¬нҳ„н•Ё.

    • мӮ¬мӢӨмғҒ м •мҲҳ(int)лӮҳ мӢӨмҲҳ(float)мқҳ н•ңк°Җм§Җ нғҖмһ…мңјлЎң мқҙлЈЁм–ҙм§җ.

    • кі м°Ёмӣҗмқҳ arrays к°ҖлҠҘ

      Source: Medium.com


  • к°Җл №, лӢӨмқҢкіј к°ҷмқҖ н–үл ¬ м—°мӮ°мқҙ мһҲлӢӨл©ҙ,

    \(\begin{bmatrix}1 & 2 \\ 3 & 4 \\ 5 & 6 \end{bmatrix} \begin{bmatrix}2 \\ -1 \end{bmatrix} = \begin{bmatrix}0 \\ 2 \\ 4 \end{bmatrix}\)

A = np.array([[1, 2],
              [3, 4],
              [5, 6]]) # 3x2 matrix
X = np.array([[2],
              [-1]]) # 2x1 matrix

A @ X  # A * X : matrix multiplication
array([[0],
       [2],
       [4]])
A.dot(X)
array([[0],
       [2],
       [4]])

DataFrame

к°Ғ м№јлҹјл“Өмқҙ н•ң к°Җм§Җ лҚ°мқҙн„° нғҖмһ…мңјлЎң мқҙлЈЁм–ҙм§„ tabularнҳ•нғң (2м°Ёмӣҗ)мқҳ лҚ°мқҙн„° нҸ¬л§·

  • NumPyмқҳ 2м°Ёмӣҗ arrayмқҳ к°Ғ м№јлҹјм—җ labelsмқ„ л¶Җм—¬н•ң кІғмңјлЎң ліј мҲҳлҸ„ мһҲмңјлӮҳ, м—¬лҹ¬ лӢӨлҘё кё°лҠҘл“Өмқҙ м¶”к°ҖлҗЁ

Missing

NaN, NA, None

pandasм—җм„ңлҠ” missingмқ„ лӘ…лӘ…н•ҳлҠ”лҚ° Rмқҳ м»ЁлІӨм…ҳмқ„ л”°лқј NA (not available)лқј л¶ҖлҰ„.
лҢҖл¶Җл¶„мқҳ кІҪмҡ°м—җм„ң NumPy object NaN(np.nan)мқ„ NAмқ„ лӮҳнғҖлӮҙлҠ”лҚ° мӮ¬мҡ©лҗЁ.
np.nanмқҖ мӢӨм ңлЎң floating-pointмқҳ нҠ№м • к°’мңјлЎң float64 лҚ°мқҙн„° нғҖмһ…мһ„. Integer лҳҗлҠ” string typeм—җм„ң м•Ҫк°„ мқҙмғҒн•ҳкІҢ мһ‘лҸҷлҗ  мҲҳ мһҲмқҢ.
Python objectмқё NoneмқҖ pandasм—җм„ң NAлЎң мқёмӢқн•Ё.
нҳ„мһ¬ NAлқјлҠ” мғҲлЎңмҡҙ pandas object мӢӨн—ҳмӨ‘мһ„
