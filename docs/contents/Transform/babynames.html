  • grouped object에 인기 상위 50% 이름의 개수를 구하는 함수를 세워 apply로 적용해 보세요.

  • 이를 이용해 아래와 같은 시각화를 해보세요.
code 
def get_quantile_count(group, q=0.5):
    prop_cumsum = group["prop"].sort_values(ascending=False).cumsum()
    return prop_cumsum[prop_cumsum <= q].size + 1

diversity = top1000.groupby(["year", "sex"]).apply(get_quantile_count)
diversity = diversity.reset_index(name="count")

  • 우선, filtered 데이터에서 연도별, 성별에 따른 출생아수의 합를 구한 후
  • 이를 연도에 따른 비율을 계산하면, 각 연도에서 Leslie와 같은 이름들이 남자로 쓰인 비율과 여자로 쓰인 비율을 구할 수 있습니다.
  • 즉, 아래 플랏에서 보면 각 연도에서 남자로 쓰인 비율과 여자로 쓰인 비율의 합이 1이 됩니다.
code 
# 연도별, 성별에 따른 출생아수의 합
table = filtered.groupby(["year", "sex"])["births"].sum().reset_index()

# 긱 연도에 따른 비율
table = table.groupby("year", group_keys=False).apply(add_prop)

# Visualization
(
    so.Plot(table, x='year', y='prop', color="sex")
    .add(so.Line())
