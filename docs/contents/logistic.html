Ordinary least squares (OLS)

  • Mean function: \(E(Y | X = x_i) = \beta_0 + \beta_1 x_i\)
  • Variance function: \(Var(Y | X = x_i) = \sigma^2\)
  • Distribution: \((Y | X = x_i) \sim N(\beta_0 + \beta_1 x_i, \sigma^2)\)


Source: p.107, Applied Regression Analysis and Generalized Linear Models (3e), by John Fox


Source: p.482, Applied Multiple Regression/Correlation Analysis for the Behavioral Sciences (3e), by Cohen, J., Cohen, P., West, S. G., & Aiken, L. S.

  • мһ”м°Ёмқҳ л¶„мӮ°мқҙ xк°’м—җ л”°лқј лӢ¬лқјм ё лӘЁм§‘лӢЁм—җ лҢҖн•ң м¶”лЎ мқ„ м–ҙл өкІҢ н•Ё.

мқҙлҹ° л¬ём ңл“Өмқ„ н•ҙкІ°н•ҳкі  мҳҲмёЎк°’мқҙ л¶„лӘ…н•ң вҖңнҷ•лҘ вҖқмқҳ мқҳлҜёлҘј н’ҲлҸ„лЎқ м—¬лҹ¬ л°©мӢқмқҙ м ңмӢңлҗҳлҠ”лҚ° мЈјлЎң мӮ¬мҡ©лҗҳлҠ” кІғмқҙ logistic regressionмһ„.

Important

Binary outcomeмқ„ мҳҲмёЎн•ҳлҠ” лӘЁнҳ•мқҖ binary к°’мқ„ мҳҲмёЎн•ҳлҠ” кІғмқҙ м•„лӢҲкі , нҷ•лҘ  к°’мқ„ мҳҲмёЎн•ҳлҠ” кІғмһ„.
мқҙнӣ„м—җ мқҙлҘј мқҙмҡ©н•ҙ binary outcomeмқ„ мҳҲмёЎ.

  • мҳҲлҘј л“Өм–ҙ, л‘җк»ҳк°Җ 5cmмқё (нҠ№м • мў…мқҳ) лӮҳл¬ҙк°Җ нғңн’Қм—җ м“°лҹ¬м§Ҳ нҷ•лҘ (true probability)мқ„ нҢҢм•…н•ҳкі мһҗ н•Ё.
  • мқҙ л•Ң, кҙҖмёЎк°’мқҖ 5cmмқё лӮҳл¬ҙ мӨ‘ м“°лҹ¬м§„ лӮҳл¬ҙмқҳ вҖңл№„мңЁвҖқмқҙкі , мқҙ кҙҖмёЎм№ҳл“ӨлЎңл¶Җн„° true probabilityлҘј м¶”м •н•ҳкі мһҗ н•Ё.

Oddsмқҳ м •мқҳ: мӢӨнҢЁн•  нҷ•лҘ  лҢҖл№„ м„ұкіөн•  нҷ•лҘ мқҳ л№„мңЁ

\(\displaystyle odds = \frac{p}{1-p}\)

мҳҲлҘј л“Өм–ҙ, 5cm л‘җк»ҳмқҳ лӮҳл¬ҙлҠ” 90к·ёлЈЁ мӨ‘ 7к·ёлЈЁк°Җ мЈҪм—ҲмңјлҜҖлЎң 83к·ёлЈЁлҠ” мӮҙм•ҳмқҢ.
мҰү, мЈҪмқҢ:мғқмЎҙ = 7:83 \(\approx\) 1:12 мқҙкі  odds = 7/83 = 0.084; мғқмЎҙн•  к°ҖлҠҘм„ұ лҢҖл№„ мЈҪмқ„ к°ҖлҠҘм„ұмқҙ 8.4%мһ„.
нҷ•лҘ лЎң н‘ңнҳ„н•ҳл©ҙ, \(odds = \frac{\frac{7}{90}}{1 - \frac{7}{90}} = \frac{7}{90-7} = \frac{7}{83}\)

нҷ•лҘ кіј odds, logit(log odds)мқҳ кҙҖкі„

мқҙ oddsлҘј м„ нҳ•лӘЁнҳ•мңјлЎң лӮҳл¬ҙл‘җк»ҳлЎң мҳҲмёЎн•ҳл Өкі  н•ҳлҠ” кІғмқёлҚ°, мҳҲлҘј л“Өм–ҙ,

\(\widehat{odds} = b_0 + b_1 \cdot log_{2}(d)\)

odds к°’мқҳ лІ”мң„лҠ” (0, \(\infty\))мқҙлҜҖлЎң, м„ нҳ•лӘЁнҳ•мңјлЎң fitн•ҳлҠ” кІғмқҙ м Ғм Ҳн•ҳм§Җ м•ҠмқҢ.

\(\displaystyle logit(p):= log(odds) = log\left(\frac{p}{1-p}\right)\)
мқҙ л•Ң, мҳҲмёЎлӘЁнҳ•мқҖ \(\displaystyle log\left(\frac{\hat{p}}{1-\hat{p}}\right) = b_0 + b_1 \cdot log_{2}(d)\)

  • м•„мқҙл””м–ҙлҠ” кҙҖмёЎм№ҳк°Җ м „мІҙм ҒмңјлЎң кҙҖм°°лҗ  likelihoodк°Җ мөңлҢҖк°Җ лҗҳлҸ„лЎқ \(b_0, b_1\)мқ„ м„ нғқн•ҳлҠ” кІғмһ„
  • мқҙлҘј мң„н•ҙм„ң м Ғм Ҳн•ң нҷ•лҘ лӘЁнҳ•мқ„ кІ°н•©мӢңмјңм•ј н•Ё.
  • кё°ліём ҒмңјлЎң м„ нғқн•ҳлҠ” нҷ•лҘ лӘЁнҳ•мқҖ binominal distribution (мқҙн•ӯл¶„нҸ¬)мһ„.
  • мқҙн•ӯл¶„нҸ¬мқҳ нҸүк· кіј н‘ңмӨҖнҺём°ЁлҠ” нҷ•лҘ  pмҷҖ nм—җ мқҳн•ҙ л°”лҖң.
  • кҙҖм°°к°’мқҖ мқҙн•ӯл¶„нҸ¬лЎңл¶Җн„° л°ңмғқн–ҲлӢӨкі  к°Җм •н•ЁмңјлЎңмҚЁ, мӢӨм ң кҙҖм°°к°’л“Өмқҳ randomness(мһ”м°Ёл“Ө)мқ„ лӘЁнҳ•м—җ л°ҳмҳҒн•  мҲҳ мһҲмқҢ.

лӘЁнҳ•мқҳ нҢҢлқјлҜён„° н•ҙм„қ

Oddsмқҳ л№„мңЁ (odds ratio, OR)мқ„ нҶөн•ҙ н•ҙм„қ

  • \(\displaystyle odds: \frac{\hat{p}}{1 - \hat{p}} = e^{b_0 + b_1 \cdot x}=e^{b_0}\cdot e^{b_1 \cdot x}\)  лЎңл¶Җн„°

  • \(x\)к°Җ 1 мҰқк°Җн•  л•Ң oddsмқҳ л№„мңЁ: \(\displaystyle odds ~ratio: \frac{\frac{\hat{p_2}}{1 - \hat{p_2}}}{\frac{\hat{p_1}}{1 - \hat{p_1}}} = e^{b_1 \cdot (x+1) - b_1 \cdot x} = e^{b_1}\)

  • мҰү, \(x\)к°Җ 1 мҰқк°Җн•ҳл©ҙ, oddsк°Җ вҖңлӘҮ л°°вҖқлЎң мҰқк°Җн•ҳлҠ”м§ҖлҘј лӮҳнғҖлғ„.

  • л”°лқјм„ң, odds ratioк°Җ 1ліҙлӢӨ нҒ¬л©ҙ (\(b_1\)мқҙ м–‘мҲҳ) \(x\)к°Җ 1 мҰқк°Җн•  л•Ң, eventмқҳ oddsк°Җ м»Өм§Җл©°,

  • odds ratioк°Җ 1ліҙлӢӨ мһ‘мңјл©ҙ (\(b_1\)мқҙ мқҢмҲҳ) eventмқҳ oddsк°Җ мӨ„м–ҙл“¬.

мң„мқҳ кІҪмҡ°, \(\displaystyle \widehat{odds} = e^{-7.82 + 2.24 \cdot log_2(d)}\) мқҙлҜҖлЎң odds ratio = \(e^{2.24} = 9.4\)

  • н•ҙм„қн•ҳл©ҙ, лӮҳл¬ҙмқҳ л‘җк»ҳк°Җ (log2 scaleлЎң) 1 лҠҳм–ҙлӮЁ (2л°° мҰқк°Җ)м—җ л”°лқј нғңн’Қм—җ лӮҳл¬ҙк°Җ м“°лҹ¬м§Ҳ oddsк°Җ 9.4л°° мҰқк°Җн•Ё
  • лӢӨмӢң л§җн•ҳл©ҙ, лӮҳл¬ҙмқҳ л‘җк»ҳк°Җ (log2 scaleлЎң) 1 лҠҳм–ҙлӮЁ (2л°° мҰқк°Җ)м—җ л”°лқј нғңн’Қм—җ лӮҳл¬ҙк°Җ м“°лҹ¬м§Җм§Җ м•Ҡмқ„ к°ҖлҠҘм„ұ лҢҖл№„ м“°лҹ¬м§Ҳ к°ҖлҠҘм„ұмқҙ 9.4л°° мҰқк°Җн•Ё.
  • лӮҳл¬ҙмқҳ л‘җк»ҳ (мӣҗлһҳ d)лЎң л§җн•ҳл©ҙ, л‘җк»ҳк°Җ 10% л‘җкәјмӣҢм§Җл©ҙ, \(e^{2.24 \cdot log_2(1.1)}=1.36\) л°°, мҰү oddsк°Җ 36% мҰқк°Җн•Ё.

Evaluattion of predicted probability

Important

мқҙм ң мқҙ лӘЁнҳ•мқҙ мўӢмқҖ лӘЁнҳ•мқём§Җ мӮҙнҺҙліҙкё° мң„н•ҙ residual, мһ”м°ЁлҘј мӮҙнҺҙліј мҲҳ мһҲлҠ”к°Җ?

Binomial version

  • Pearson residual: \(\displaystyle \frac{actual ~ count ~ - predicted ~ count}{SD ~ of ~ count} = \frac{y_i - m_i \hat{p}_i}{\sqrt{m_i \hat{p}_i (1-\hat{p}_i)}}\)
  • Deviance residual: \(\displaystyle sign(y_i - m_i\hat{p}_i) \sqrt{2[y_i log(\frac{y_i}{m_i\hat{p}_i}) + (m_i-y_i) log(\frac{m_i - y_i}{m_i-m_i\hat{p}_i})]}\)

Binary version

  • Pearson residual: \(\displaystyle \frac{y_i - \hat{p}_i}{\sqrt{\hat{p}_i (1-\hat{p}_i)}}\)
  • Deviance residual: \(\displaystyle sign(y_i - \hat{p}_i) \sqrt{-2[-y_i log(\hat{p}_i) - (1-y_i) log(1-\hat{p}_i)]}\)

DevianceлҘј мқҙмҡ©н•ҙ OLSм—җм„ңмқҳ \(R^2\)мҷҖ л№„мҠ·н•ң к°ңл…җмқ„ кө¬м„ұ

  • Cox-Snell \(R^2\)
  • Nagelkerke \(R^2\)

вҖңCoefficient of discriminationвҖқ (Tjur, 2009): average \(\hat{p}\) when \(y=1\) - average \(\hat{p}\) when \(y=0\)

Logistic Regression м •лҰ¬

Binary responseм—җ лҢҖн•ҙ к·ёл“Өмқҳ conditional нҸүк· мқё нҷ•лҘ  \(p(x_i)\)лҘј м¶”м •

Distribution: \((Y_i | X = x_i) \sim Bin(m_i, p(x_i))\) мқҳ л¶„нҸ¬к°Җ binomial distributionмқ„ л”°лҰ„. л”°лқјм„ң \(Y_i\)мқҳ нҸүк· кіј л¶„мӮ°мқҖ \(m_i\)мҷҖ \(p(x_i)\)м—җ л”°лқј кІ°м •лҗҳм–ҙ ліҖн•Ё.

  • лӮҳл¬ҙмқҳ л‘җк»ҳ(d)лҘј к°Җм§„ mк°ңмқҳ лӮҳл¬ҙл“Өм—җ лҢҖн•ҙ нғңн’Қм—җ м“°лҹ¬м§ҖлҠ” лӮҳл¬ҙмқҳ мҲҳ(count)лҠ” мқҙн•ӯл¶„нҸ¬ \(Bin(m, p)\) лҘј л”°лҰ„.

Linear predictors: нҷ•лҘ  \(p(x_i)\)лҠ” link function (мң„м—җм„ңлҠ” logit)лҘј нҶөн•ҙ мҳҲмёЎліҖмҲҳл“Өмқҳ м„ нҳ• кІ°н•©мңјлЎң кІ°м •лҗЁ.

  • \(\displaystyle log\left(\frac{\hat{p}}{1-\hat{p}}\right) = b_0 + b_1 \cdot X_1 + b_2 \cdot X_2\)

  • Probit link functionмңјлЎң м“ё мҲҳлҸ„ мһҲмқҢ: \(\displaystyle \Phi^{-1}(\hat{p}) = b_0 + b_1 \cdot X_1 + b_2 \cdot X_2\)

    • \(\Phi\): cumulative standard normal distribution function

мң„мқҳ м•„мқҙл””м–ҙлҠ” generalized linear model (GLM)лЎң нҷ•мһҘлҗҳлҠ”лҚ°,

  • \((Y_i | X = x_i)\)мқҳ л¶„нҸ¬к°Җ exponential family distributionмқ„ л”°лҰ„
  • \(Y\)лҠ” мҳҲмёЎліҖмҲҳл“Өмқҳ м„ нҳ• кІ°н•©мңјлЎңл§Ң кІ°м •лҗҳл©°,
  • Link functionмқ„ нҶөн•ҙ \(Y_i\)мқҳ нҸүк· мқ„ мҳҲмёЎн•Ё. мҰү, \(g(E(Y | X=x_i)) = \beta_0 + \beta_1 \cdot x_i\)
